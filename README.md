## What's is...
English Translation For TOS Thai Server.
Base on English Steam.

### Language
* Thi คือ ภาษาไทยต้นฉบับ
* Thai English คือ ภาษาอังกฤษ ผสานกับภาษาไทยบางส่วนที่เป็นของเดิม
* English Steam คือ ภาษาอังกฤษต้นฉบับของสตีม
